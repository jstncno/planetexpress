PlanetExpress
=============
PlanetExpress is my custom Network Attached Storage (NAS), developed on the [Banana Pi](http://www.banana-pi.org).

# Features

* NFS mounted server
* Music organized by [Beets](https://github.com/sampsyo/beets)
